About

The purpose of this extension is to help you read faster. It achieves that by displaying the text from the web page you are reading word by word really fast. With a little training it is possible to achieve 500 - 600 words per minute. Achieving such speeds is thanks to Reading Helper eliminating the need to move your eyes left to right, in addition to keeping your eyes focused, instead of them wandering and going backwards.

Settings

Keepin' it real

Below are listed the controls and instruction for using Reading Helper

[Ctrl + Shift + Y] - Start of setup for new websites.

[Ctrl + Shift + K] - Start reading.

[Ctrl + Shift + Q] - End reading.

When you open new website, in order to start reading with Reading Helper you need to do the initial setup for that website. First press the command for start of setup, then you need to double-click on a word from two separate paragraphs. After you do that you can start reading. This setup need to be done only once for every website you want to read use Reading Helper. You can always refresh the setup for that website if for some reason Reading Helper is not displaying the correct text. To redo the setup you need to do the steps for the initial setup.

When you are ready with the setup you can start reading.
If for some reason you want to stop reading you can press the command for End reading or simply refresh the page.
